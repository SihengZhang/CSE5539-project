# Section 1: Problem Statement - Working Notes

## Key Points to Cover

### The Problem
- Scientific simulations generate petabytes of data
- Storage, transmission, and analysis costs are prohibitive
- Current compression treats all data uniformly

### Why It Matters
- Critical topological features (minima, maxima, saddles) are essential for science
- Uniform compression can destroy or displace these features
- Scientists need topology-preserving compression

### Who Cares
- Computational scientists running large simulations
- Data centers storing scientific archives
- Researchers sharing data across institutions
